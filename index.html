How to upload html to Github

  1. Open Terminal
  2. Use the cd command to change directories into your repo folder
  3. Turn this folder into a git repo
  4. Use the following command:
  5. git init
  6. You should see the message “Initialized empty Git repository”
  7. Then use git add
  8. git add README.md
  9. You can run git status to see added files
  10. Next you need to commit changes
  11. Commit early and commit often.
  12. git commit -am "first commit"
  13. Now we need to change to the main branch:
  14. git branch -M main
  15. We need to add the remote repo as the “origin” of this branch.
  16. Copy this line from GitHub: git remote add origin (link)
  17. Make sure you get the correct version directly from GitHub.
  18. Now you need to push your changes to GitHub.
  19. git push origin main
  20. If there are no errors, load the repo in your browser and see if you see your code appears.
